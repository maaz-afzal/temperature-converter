# Temperature Converter

A simple web app that converts temperatures between Celsius, Fahrenheit, and Kelvin in real time.


## Features

- Type in any field and the other two update instantly
- Supports Celsius, Fahrenheit, and Kelvin
- Reset button to clear all fields

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run

No installation needed. Just follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/maaz-afzal/temperature-converter.git
   ```

2. Go into the project folder:
   ```
   cd temperature-converter
   ```

3. Open `index.html` in your browser and you are good to go.
